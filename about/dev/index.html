Adding Sources Using Query Parameter

JSON = encodeURIComponent(JSON.stringify({
  // Object contents depends on type of source, see below.
  kind: "ipfs",
  data: {
    name: "Music from IPFS",

    // Source type specific
    directoryHash: "Qm..."
  }
}))

"https://diffuse.sh?source=JSON"

You can add multiple “source” query parameters, if you want to add multiple sources.

Amazon S3

kind:
  amazon_s3 

data:
  # Required
  accessKey
  bucketName
  name
  region
  secretKey

  # Optional
  directoryPath
  host

Azure

kind:
  "azure_file" # or "azure_blob" 

data:
  # Required
  accountName
  accountKey
  container
  name

  # Optional
  directoryPath

BTFS

kind:
  btfs 

data:
  # Required
  directoryHash
  name

  # Optional
  gateway

Dropbox

kind:
  dropbox 

data:
  # Required
  accessToken
  appKey
  name

  # Optional
  directoryPath

Google

kind:
  google 

data:
  # Required
  authCode
  clientId
  clientSecret
  name

  # Optional
  folderId

IPFS

kind:
  ipfs 

data:
  # Required
  directoryHash
  name

  # Optional
  gateway
  ipns              ← boolean, `t` of `f`
  local             ← boolean, `t` of `f`

WebDAV

kind:
  webdav 

data:
  # Required
  name
  url

  # Optional
  directoryPath
  password
  username
